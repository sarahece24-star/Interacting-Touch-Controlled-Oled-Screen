* Arduino-based handheld OLED mini console using a 1.3-inch SH1106 display
* Features a custom menu-driven user interface controlled using push buttons
* Includes an animated pixel-art home screen with frame-based animation
* Implements a Dino-style endless runner game inspired by the Chrome offline game
* Player controls a jumping character to avoid moving cactus obstacles
* Uses real-time jump physics, gravity simulation, and collision detection
* Supports score tracking and increasing game difficulty over time
* Includes multiple system states such as Home Screen, Menu, Game, Timer, and Game Over screens
* Built using Adafruit GFX and SH110X graphics libraries for OLED rendering
* Uses debounce logic for stable button input handling
* Designed and tested on breadboard before final vero board implementation
* Combines embedded systems programming, OLED graphics, hardware interfacing, and game logic development
* Beginner-friendly ECE mini project integrating both hardware and software concepts
